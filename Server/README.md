## Run on Local

```
git clone https://github.com/liara-cloud/hono-getting-started.git
```
```
cd hono-getting-started
```
```
npm install
```
```
npm run dev
```

## Deploy on Liara
```
git clone https://github.com/liara-cloud/hono-getting-started.git
```
```
cd hono-getting-started
```
```
liara deploy --platform node --port 3000
```
